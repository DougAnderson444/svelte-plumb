# Svelte Plumb

Going for the simplest svg drawing API ever.

```gfm
<Canvas let:connectable>

    <div use:connectable>
        Svelte action directive now applied to this HTMLElement
    </div>

    <div use:connectable>
        Svelte action directive now applied to this HTMLElement
    </div>

</Canvas>
```
